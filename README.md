# VehicleDismountDisabler

A Spigot plugin that prevent players to exit certain vehicles in certain worlds.

To get out of a vehicle in a blacklisted world, the vehicle has to be destroyed.

## Valid vehicles
 - MINECART
 - BOAT
 - PIG
 - HORSE
 - SKELETON_HORSE
 - ZOMBIE_HORSE
 - DONKEY
 - MULE
 - LLAMA
